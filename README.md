## 📌 About

This repository contains Kotlin programs and practice files created while learning Kotlin programming and exploring modern development concepts.

Topics covered include:
- Kotlin Fundamentals
- Object-Oriented Programming (OOP)
- Functions & Lambda Expressions
- Collections Framework
- Exception Handling
- Generics
- Coroutines & Threads

The repository is focused on strengthening programming fundamentals and building a solid base for Android Development.
